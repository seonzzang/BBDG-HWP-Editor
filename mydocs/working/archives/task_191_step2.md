# 타스크 191 2단계 완료 보고서: 표·셀·여백캡션 탭 고도화

## 완료 항목

### 1. 표 탭 보강
- **쪽 경계에서(Q)**: 라디오 2종 → 드롭다운 3종 (`나눔`/`셀 단위로 나눔`/`나누지 않음`)
  - value: 2=나눔, 1=셀 단위로 나눔, 0=나누지 않음
- **자동으로 나뉜 표의 경계선 설정(J)**: 체크박스 + 종류(N)/굵기(H)/색(S) 하위 필드
  - 체크 시 하위 필드 활성화 (css `tcp-disabled` 토글)
- **"모두(A)" 스피너**: 모든 셀 안 여백 일괄 조정 (▲/▼ 버튼으로 0.5mm 단위 증감)

### 2. 셀 탭 보강
- **"모두(A)" 스피너**: 안 여백 일괄 조정 (표 탭과 동일한 `buildAllSpinner` 재사용)
- **"한 줄로 입력(S)"**: `disabled` 해제 → 활성 상태
- **세로쓰기 하위 옵션**: "문 눕힘(Q)/문 세움(U)" 2종 버튼 추가
  - 세로쓰기 버튼 클릭 시 하위 옵션 활성화, 가로쓰기 시 비활성 (`tcp-disabled`)

### 3. 여백/캡션 탭 보강
- **"모두(A)" 스피너**: 바깥 여백 일괄 조정
- **캡션 넣기 아이콘 그리드**: 드롭다운 → 3×3 시각적 SVG 아이콘 (8종 위치)
  - 위/아래/왼쪽(위·가운데·아래)/오른쪽(위·가운데·아래)
  - CSS: `tcp-caption-grid` + `tcp-caption-item` 클래스 활용
  - 내부 `captionDirSelect`는 숨겨진 상태로 값 관리
- **캡션 크기(S)**: mm 스피너 입력 추가
- **여백 부분까지 너비 확대(W)**: 체크박스 추가

### 4. 공통 컴포넌트
- `buildAllSpinner()` 헬퍼 메서드: 4방향 여백 입력을 일괄 조정하는 재사용 가능 위젯
  - ▲/▼ 버튼으로 모든 방향 동시 0.5mm 증감
  - 4방향 값이 모두 같으면 수치 표시, 다르면 "—" 표시
  - 표 탭, 셀 탭, 여백/캡션 탭에서 공통 사용
