# 타스크 191 3단계 완료 보고서: 표 테두리·배경 탭 고도화

## 완료 항목

### 1. 테두리 탭 전면 개편

#### 선 종류 시각적 격자
- `<select>` 드롭다운 → SVG 아이콘 기반 시각적 격자 (`tcp-line-type-grid`)
- 8종 선 유형: 없음/실선/파선/점선/일점쇄선/이점쇄선/긴 파선/이중 실선
- 각 항목은 SVG `<line>` 요소로 선 샘플 표시, 클릭 시 `active` 클래스 토글
- `borderSelectedLineType` 필드로 현재 선택된 선 종류 추적

#### SVG 기반 십자선 미리보기
- CSS border 기반 → SVG 기반 미리보기로 전면 교체
- viewBox `0 0 120 100`, 연한 회색 점선 십자선(셀 구분선) 포함
- 4방향 테두리를 SVG `<line>` 요소로 렌더링 (선 종류별 dasharray 매핑)
- 이중 실선은 2개 line을 offset으로 렌더링

#### 방향 버튼 그리드 배치
- 기존 일렬 버튼 → `tcp-border-preview-wrap` 그리드 배치 (3×3)
- 모두(O): 좌상, 위(▲): 상중, 왼(◀): 좌중, SVG: 중앙, 오른(▶): 우중, 아래(▼): 하중

#### 자동 경계선 설정 + 안내 문구
- "자동으로 나뉜 표의 경계선 설정(J)" 체크박스 + 종류/굵기/색 하위 필드
- "※ 표 테두리는 [셀 간격]에 값을 입력해야 나타납니다" 안내 문구 추가

#### 선 모양 바로 적용
- "선 모양 바로 적용(I)" 체크박스 추가

### 2. 배경 탭 고도화

#### 무늬 채우기 3개 필드 활성화
- **면색(C)**: 기존 컬러 피커
- **무늬색(K)**: 신규 컬러 피커 추가
- **무늬모양(L)**: 드롭다운 7종 (없음/가로줄/세로줄/역슬래시/슬래시/십자/X자)

#### 미리보기 무늬 패턴 렌더링
- CSS `repeating-linear-gradient` 기반 무늬 프리뷰
- 면색 위에 무늬색 패턴 오버레이 표시

#### 속성 저장
- `onConfirm()`에서 `patternColor`, `patternType` 속성 저장
- `populateBgFromTarget()`에서 기존 무늬 속성 불러오기
