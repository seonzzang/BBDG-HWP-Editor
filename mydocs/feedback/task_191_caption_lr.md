# 피드백: 표 캡션 좌/우 배치 동작 (타스크 191 관련)

## 한컴 동작 확인 결과

### 좌/우 캡션 배치 규칙

| 캡션 위치 | 표 위치 변화 | 캡션 배치 |
|-----------|-------------|-----------|
| 왼쪽 (위/가운데/아래) | 표가 캡션 크기만큼 오른쪽으로 밀림 | 표 왼쪽에 캡션 배치 |
| 오른쪽 (위/가운데/아래) | 표 위치 변경 없음 | 표 오른쪽에 캡션 배치 |
| 위/아래 | 표 위치 변경 없음 | 표 위/아래에 캡션 배치, 캡션 크기 설정 불필요 |

### 세부 동작

- 왼쪽 캡션 30mm 설정 시: 왼쪽 30mm 이후에 표가 렌더링됨
- 오른쪽 캡션 30mm 설정 시: 표 렌더링 위치는 변경 없이 표 오른쪽에 캡션 배치
- 캡션 내용이 캡션 크기를 초과하면 줄넘김 처리
- "한 줄로 입력" 속성 지정 시 줄넘김 하지 않음

## 구현 필요 사항

1. `captionWidth` → Rust IR (`cap.width`) 저장
2. Left 캡션: 표 x 좌표를 `cap.width`만큼 오른쪽으로 이동, 캡션은 원래 x에 배치
3. Right 캡션: 표 x 좌표 유지, 캡션은 표 오른쪽 끝 + spacing에 배치
4. 캡션 폭 기반 동적 줄넘김 (compose_paragraph 또는 layout 단계)
5. 한줄 입력 속성 지원
