# 타스크 189: 이미지 컨트롤 속성 UI 고도화

## 개요

이미지(Picture) 컨트롤 선택 후 컨텍스트 메뉴 → "개체 속성" 대화상자의 속성 바인딩 및 세부 속성 변경 반영을 완성한다.

## 현재 상태 분석

### 대화상자 탭 구성 (이미지용)

| 탭 | 상태 | 설명 |
|----|------|------|
| 기본 | 부분 완성 | 크기/위치/배치 동작. 회전·대칭·기울이기 비활성 |
| 여백/캡션 | 부분 완성 | 여백 동작. 캡션 크기·간격·옵션 비활성 |
| 선 | 완성 | 테두리 선 속성 |
| 그림 | **미완성** | stub panel ("추후 구현 예정") |
| 그림자 | 완성 | 그림자 속성 |

### 주요 미완성 항목

1. **"그림" 탭 미구현**: 밝기, 명도(contrast), 효과(effect) UI가 없음
2. **기본 탭 비활성 기능**: 회전, 좌우/상하 대칭, 기울이기, 쪽 영역 제한, 겹침 허용, 같은 쪽에 놓기
3. **여백/캡션 탭**: 캡션 세부 설정 비활성
4. **Rust 속성 수정 API**: brightness/contrast만 지원, effect 미지원

## 수행 범위

### 우선순위 높음 (이번 타스크)
- "그림" 탭 구현 (밝기, 명도, 효과)
- 기본 탭 비활성 기능 중 회전·대칭 활성화
- Rust 측 effect 속성 지원 추가

### 우선순위 낮음 (향후)
- 기울이기, 쪽 영역 제한 등 고급 배치 옵션
- 캡션 세부 설정
- 그림 채우기 (워터마크 등)
