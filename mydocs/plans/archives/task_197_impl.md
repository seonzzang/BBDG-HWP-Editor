# 타스크 197 구현계획서 — 다양한 줄간격 문서의 페이지 넘김 검증

## 단계별 구현 계획

### 1단계: 네이티브 단위 테스트 작성

- 다양한 줄간격(100%, 160%, 250%, 300%)으로 문단을 생성하고 페이지 넘김 위치 검증
- 혼합 줄간격 문서에서 페이지 수와 각 페이지의 문단 배치 검증
- Fixed/SpaceOnly/Minimum 타입 줄간격 테스트
- `cargo test`로 검증

### 2단계: E2E 줄간격 변경 + 페이지 넘김 브라우저 테스트

- `applyParaFormat` API로 문단별 줄간격 변경
- 문단부호 ON 상태에서 줄간격 변화에 따른 시각적 레이아웃 검증
- 페이지 넘김 시점이 줄간격에 따라 정확히 달라지는지 확인
- 스크린샷 캡처 + pageCount 검증

### 3단계: 버그 수정 및 최종 검증

- 1~2단계에서 발견된 문제 수정
- 전체 테스트 통과 확인 (네이티브 + E2E)
- 최종 결과보고서 작성
