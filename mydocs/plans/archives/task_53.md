# 타스크 53 수행계획서: 하단 상태 바 (Status Bar) UI 구현

## 목표

한컴 웹기안기와 동일한 하단 상태 바 UI를 구현한다.

## 현재 상태

- 줌 컨트롤과 페이지 정보가 서식 바(`#style-bar`) 우측에 위치
- 상태 메시지가 아이콘 툴바(`#icon-toolbar`) 우측에 위치
- 전용 하단 상태 바 없음

## 구현 범위

- 화면 최하단에 한컴 웹기안기와 동일한 상태 바 추가
- 좌측: 페이지 정보, 구역 정보, 삽입/수정 모드
- 우측: 줌 맞춤 아이콘, 줌 비율, 줌 축소/확대 아이콘
- 한컴 SVG 스프라이트 아이콘 사용 (줌 관련 row 24)
- 기존 서식 바에서 줌/페이지 요소 제거

## 수정 파일

| 파일 | 변경 |
|------|------|
| `rhwp-studio/index.html` | 상태 바 HTML 추가, 서식 바 줌/페이지 제거 |
| `rhwp-studio/src/style.css` | 상태 바 CSS, 줌 스프라이트 아이콘 |
| `rhwp-studio/src/main.ts` | 이벤트 리스너 이전, 줌 맞춤, 상태 메시지 |
| `rhwp-studio/src/engine/input-handler.ts` | Insert 키 모드 토글 |

## 검증

- Vite 빌드 성공
- 런타임: 하단 상태 바 표시, 줌/페이지 동작 확인
