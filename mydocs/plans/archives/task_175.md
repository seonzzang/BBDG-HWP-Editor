# 타스크 175: 환경설정 대화상자 + 대표글꼴

## 배경

B-011에서 등록된 요구사항. 한컴 한/글의 "도구 > 환경설정 > 글꼴" 탭 기능을 rhwp-studio에 구현한다.

### 핵심 요구사항

1. **확장 가능한 사용자 설정 저장 구조** - 대표글꼴뿐 아니라 향후 다양한 환경 설정을 수용
2. **환경설정 대화상자** - 도구 메뉴에서 접근, 탭 구조 (글꼴 탭 우선)
3. **대표 글꼴 세트** - 7개 언어별(한글/영문/한자/일어/외국어/기호/사용자) 글꼴 프리셋 등록/편집/삭제

### 한컴 도움말 분석 결과

- **대표 글꼴**: 응용프로그램 수준 설정 (문서별 아님), `Fontinfo.ini`에 저장
- **글꼴 탭 구성**: 글꼴 보기(미리보기, 최근 사용) + 대표 글꼴 등록 + 글꼴 검색
- **대표 글꼴 대화상자**: 목록 + 추가/편집/삭제 아이콘 → 하위 대화상자(이름 + 7개 언어 글꼴)
- 웹 환경에서는 **localStorage** 사용 (확장 가능한 키 구조)

### 현재 코드베이스 상태

- 설정 저장 인프라 **없음** (localStorage/IndexedDB 미사용)
- "도구" 메뉴 **미존재** (HTML에 추가 필요)
- 글꼴 목록 **하드코딩** (toolbar, CharShapeDialog 모두 정적 배열)
- ModalDialog 기반 대화상자 패턴 확립 (dialog.ts)
- CSS 접두어 규칙: `dialog-` (공통), 대화상자별 접두어 (cs-, ps-, sd-)

## 수행 범위

### 포함

1. 사용자 설정 저장/로드 서비스 (UserSettings)
2. "도구" 메뉴 추가 + "환경설정" 메뉴 아이템
3. 환경설정 대화상자 (탭 구조, 글꼴 탭)
4. 대표 글꼴 등록 대화상자 (추가/편집/삭제)
5. 서식바 글꼴 드롭다운에 대표 글꼴 반영

### 제외 (향후 별도 타스크)

- 글꼴 미리보기 (글꼴 이름을 해당 글꼴로 렌더링)
- 자주 사용하는 글꼴 설정
- 글꼴 목록 탭별 표시
- 편집/기타/파일 탭 등 환경설정 나머지 탭

## 기술 설계

### 1. UserSettings 서비스 (localStorage 기반)

```
localStorage key: "rhwp-settings"
값: JSON 문자열

{
  "version": 1,
  "font": {
    "fontSets": [
      {
        "name": "사용자 세트 1",
        "korean": "함초롬바탕",
        "english": "Times New Roman",
        "chinese": "함초롬바탕",
        "japanese": "함초롬바탕",
        "other": "함초롬바탕",
        "symbol": "함초롬바탕",
        "user": "함초롬바탕"
      }
    ],
    "recentFontCount": 3,
    "showFontPreview": false
  },
  "edit": {},
  "view": {}
}
```

- 단일 키에 전체 설정을 JSON으로 저장 → 확장 시 섹션 추가만 하면 됨
- `version` 필드로 마이그레이션 지원
- 읽기/쓰기 API: `UserSettings.get(path)`, `UserSettings.set(path, value)`

### 2. 기본 대표 글꼴 프리셋 (내장)

한컴 기본 제공 대표 글꼴에 해당하는 내장 프리셋:

| 이름 | 한글 | 영문 |
|------|------|------|
| 함초롬 | 함초롬바탕 | 함초롬바탕 |
| 돋움 | 함초롬돋움 | 함초롬돋움 |
| 맑은 고딕 | 맑은 고딕 | 맑은 고딕 |
| 바탕 | 바탕 | 바탕 |

사용자 정의 세트는 localStorage에 저장.

### 3. UI 구조

```
도구 메뉴 > 환경설정
  → OptionsDialog (탭 구조)
    → [글꼴] 탭
      → 글꼴 보기 섹션 (최근 사용 글꼴 개수)
      → 대표 글꼴 등록 섹션
        → [대표 글꼴 등록하기] 버튼
          → FontSetDialog (목록 + 추가/편집/삭제)
            → FontSetEditDialog (이름 + 7개 언어 글꼴 선택)
```

### 4. CSS 접두어

| 접두어 | 대상 |
|--------|------|
| `opt-` | 환경설정 대화상자 (OptionsDialog) |
| `fs-` | 대표 글꼴 등록 대화상자 (FontSetDialog) |
| `fse-` | 대표 글꼴 편집 대화상자 (FontSetEditDialog) |

## 위험 요소

- 웹 글꼴 목록을 시스템 설치 글꼴처럼 동적으로 가져올 수 없음 → 등록된 웹 글꼴 + font-loader.ts 목록 기반으로 제공
- localStorage 용량 제한 (5MB) → 글꼴 설정 정도는 충분
