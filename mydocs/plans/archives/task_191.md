# 타스크 191: 표 설정창 UI 고도화

## 현황 분석

현재 `table-cell-props-dialog.ts` (1095줄)에 6탭(기본/여백·캡션/테두리/배경/표/셀) 구현 완료 상태.
한컴 실제 대화상자 스크린샷 7장과 비교하여 아래 Gap을 도출하였다.

### 핵심 구조적 차이

한컴은 **문맥에 따라 대화상자 구성이 달라진다:**

| 선택 상태 | 대화상자 | 탭 구성 |
|-----------|----------|---------|
| 표 선택 (개체 선택) | 표/셀 속성 | 기본 / 여백·캡션 / **테두리** / **배경** / 표 / 셀 (6탭) |
| 셀 선택 (셀 내 커서/셀 블록) | 표/셀 속성 | 기본 / 여백·캡션 / 표 / 셀 (**4탭**, 테두리·배경 없음) |
| 셀 블록 → 우클릭 "셀 테두리/배경" | **셀 테두리/배경** (별도 대화상자) | 테두리 / 배경 / 대각선 (3탭) |

셀 테두리/배경 대화상자는 하위 메뉴로 접근:
- **각 셀마다 적용(E)...** — 선택된 각 셀에 개별 적용
- **하나의 셀처럼 적용(Z)...** — 선택 범위를 단일 셀 취급

현재 구현은 항상 6탭을 표시하고, 테두리/배경 탭 내 라디오로 셀/표 전환 → 한컴과 구조가 다름.

## 목표

한컴 표/셀 속성 대화상자의 **상 우선순위 항목**을 중심으로 UI를 한컴 수준으로 고도화한다.

1. **대화상자 구조 개편** — 문맥별 탭 분기 (표 선택=6탭, 셀 선택=4탭)
2. **셀 테두리/배경 별도 대화상자** 신규 생성 (테두리/배경/대각선 3탭)
3. **테두리 탭 고도화** — 선 종류 시각적 격자 + 십자선 미리보기 + 자동 경계선 설정
4. **배경 탭 고도화** — 무늬 채우기 (면색+무늬색+무늬모양) 활성화
5. **표 탭 보강** — "나눔" 3종 + 자동 경계선 설정 + "모두" 일괄 여백
6. **여백/캡션 탭 보강** — 캡션 넣기 아이콘 그리드 (8종) + "모두" 일괄 조정
7. **셀 탭 보강** — "모두" 일괄 여백 + 한 줄로 입력 + 세로쓰기 눕힘/세움
8. **전용 CSS 분리** + 시각 품질 향상
9. **컨텍스트 메뉴 연동** — "셀 테두리/배경" 하위 메뉴 (각 셀마다/하나의 셀처럼)

## 범위 제한
