# 타스크 158 수행계획서: 글상자 테두리/채우기/세로쓰기

## 타스크 개요

글상자(Shape) 개체의 테두리/채우기/세로쓰기/기호 대체/회전/도형 변환 기능을 완성한다.

## 현재 상태 분석

| 기능 | 렌더링 | UI | 비고 |
|------|--------|-----|------|
| 테두리 선 종류/굵기/색 | ✓ 완료 | ✓ 완료 | 11종 선 종류, SVG/Canvas 모두 지원 |
| 테두리 화살표 | ✓ SVG, △ Canvas | ✓ 완료 | Canvas는 마커 미지원 |
| 면 채우기 - 단색 | ✓ 완료 | ✓ 완료 | |
| 면 채우기 - 그라데이션 | ✓ 완료 | ✓ 완료 | 선형+방사형, 3~4번 타입은 근사 |
| 면 채우기 - 그림 | ✓ 완료 | ✓ 완료 | 타일/확대/중앙 등 |
| 면 채우기 - 패턴 | △ 미검증 | ✓ 완료 | 빗금 등 패턴 렌더링 확인 필요 |
| 세로쓰기 - 표 셀 | ✓ 완료 | N/A | 완전한 세로 칼럼 레이아웃 |
| 세로쓰기 - 글상자 | ✗ 미구현 | ✓ UI 준비됨 | 핵심 작업 대상 |
| 기호 대체 | ✗ 미구현 | ✗ 없음 | 세로쓰기용 구두점 회전/대체 |
| 회전 (90도 단위) | ✓ 아핀 변환 지원 | △ 입력만 존재 | UI 저장/반영 확인 필요 |
| 도형→글상자 변환 | ✗ 미구현 | ✗ 없음 | 우선순위 낮음 |
| 개체 속성 대화상자 | N/A | 85~90% 완료 | 일부 컨트롤 비활성 |

## 핵심 작업 범위

### 반드시 구현할 항목 (Must)
1. **글상자 세로쓰기**: 표 셀의 세로쓰기 로직을 글상자에 통합
2. **기호 대체**: 세로쓰기 시 구두점 방향 회전/대체 (마침표→고리점 등)
3. **개체 속성 대화상자 연동**: 세로쓰기/텍스트 방향 설정이 실제 렌더링에 반영되도록

### 보완할 항목 (Should)
4. **회전 속성 UI 연동**: 대화상자에서 회전 각도 설정 → 저장 → 렌더링 반영
5. **패턴 채우기 렌더링 검증**: 빗금/격자 등 패턴이 실제 렌더링되는지 확인
6. **비활성 UI 컨트롤 활성화**: 선 투명도, 그림자 방향 등 disabled 컨트롤 처리

### 후순위 항목 (Could)
7. **도형→글상자 변환**: 도형 안에 글자 넣기/글상자 속성 없애기
8. **새 그리기 속성 등록**: BorderFill 신규 등록 WASM API

## 구현 계획 (4단계)

### 1단계: 글상자 세로쓰기 렌더링
- `layout_textbox_content()`에 세로쓰기 분기 추가
- 표 셀의 `layout_vertical_cell_text()` 로직을 글상자용으로 적응
- TextBox의 `list_attr` → text_direction 비트 파싱
- 영문 눕힘(text_direction=1) / 영문 세움(text_direction=2) 모두 지원
- SVG/Canvas 렌더링 검증

### 2단계: 기호 대체 + 세로쓰기 보강
- 세로쓰기 시 구두점 방향 대체 테이블 정의 (한컴 도움말 참조)
- 마침표→고리점, 쉼표→읽기점 등 매핑
- `layout_vertical_cell_text`와 글상자 세로쓰기 공통으로 적용
- CJK 문자 판정/회전 판정 로직 보강

### 3단계: 회전/패턴 채우기/대화상자 연동
- 글상자 회전 각도 UI 설정 → WASM API → 저장/렌더링
- 패턴 채우기(빗금 등) SVG/Canvas 렌더링 구현/검증
- 대화상자 비활성 컨트롤 중 구현 가능한 항목 활성화

### 4단계: 테스트 + 회귀 검증
- 세로쓰기 샘플 HWP 파일 SVG 내보내기 검증
- 모든 608+ 테스트 통과 확인
- 122페이지 p222.hwp 회귀 테스트
- 오늘할일 상태 갱신
