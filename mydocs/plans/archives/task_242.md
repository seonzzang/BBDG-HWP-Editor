# Task 242 수행 계획서: 도형 기본 삽입 (직선/사각형/타원)

## 현재 구현 상태

### ✅ 이미 구현된 것

| 영역 | 상태 | 상세 |
|------|------|------|
| 도형 모델 | 완성 | Line, Rectangle, Ellipse, Arc, Polygon, Curve, Group |
| 파싱/직렬화 | 완성 | round-trip 정상 |
| 렌더링 | 완성 | SVG/Canvas 모두 도형 정상 표시 |
| Rectangle+TextBox 생성 | 완성 | `createShapeControl()` WASM API |
| 배치 모드 | 완성 | 드래그 오버레이 패턴 (글상자/그림에서 사용) |
| 선택/핸들 | 완성 | 8방향 리사이즈 + 회전 핸들(녹색) |
| 리사이즈/이동/회전 | 완성 | 마우스 드래그, 최소 크기 제한, 15° 스냅 |
| 속성 조회/수정 | 완성 | 50+ 속성 JSON 기반 get/set |
| Z-order | 완성 | 앞으로/뒤로/맨앞/맨뒤 |
| 삭제 | 완성 | `deleteShapeControl()` |
| 회전/대칭 | 완성 | 90° 회전, 좌우/상하 대칭 |

### ❌ 구현 필요한 것

| 영역 | 난이도 | 상세 |
|------|--------|------|
| 도형 타입 선택 UI | 낮음 | 드롭다운 패널에 아이콘 버튼 배치 |
| 다양한 도형 생성 API | 중간 | `create_shape_control_native()` 확장 (타입 파라미터) |
| 직선(Line) 생성 | 중간 | LineShape 인스턴스 + 시작/끝점 계산 |
| 타원(Ellipse) 생성 | 낮음 | Rectangle과 유사한 구조 |
| 연결선(ObjectLinkLine) | 높음 | `$col` ctrl_id, 시작/끝 개체 연결, 제어점, 자동 라우팅. 모델/파서 미구현 |
| 직선 화살표 속성 | 중간 | LineType, ArrowStart/End UI |
| 다각형/곡선 편집 | 높음 | 꼭짓점/제어점 편집 모드 |
| 그룹 개체 | 높음 | 다중 선택 + 그룹/해제 |
| 도형 내 텍스트 편집 | 중간 | TextBox 포커스 전환 (글상자와 동일) |

### 연결선 ($col, ObjectLinkLine) 분석

hwplib 기준 연결선 고유 속성:
- 시작/끝 좌표 (startX/Y, endX/Y)
- 연결선 타입 (LinkLineType)
- 시작/끝 연결 대상 (startSubjectID/Index, endSubjectID/Index)
- 제어점 목록 (ControlPoint[]) — 꺾이는 지점
- **특이사항**: 두 개체를 연결하며, 개체 이동 시 자동 추적하는 기능

우리 현재 상태:
- 모델 미정의, 파서 미구현, 렌더러 미구현
- 단순 직선과 달리 **개체 간 연결 관계**를 관리해야 하므로 난이도 높음

## 구현 수준별 분석

### Level 1: 기본 도형 삽입 (소요: 1일)

기존 인프라만으로 구현 가능. 새 코드 최소화.

- **도형 선택 드롭다운**: 직선, 사각형, 타원 3종 아이콘 버튼
- **`create_shape_control_native()` 확장**: shape_type 파라미터 추가
  - Rectangle: 기존 코드 재활용
  - Ellipse: Rectangle과 동일 구조, ShapeObject::Ellipse로 변경
  - Line: start_x/y, end_x/y 계산 추가
- **배치 모드**: 기존 textboxPlacementMode 패턴 재사용
- **`insert:shape` 커맨드 활성화**

**결과**: 3종 도형 드래그 삽입 → 선택/리사이즈/이동/회전/삭제 모두 기존 코드로 동작

### Level 2: 도형 속성 편집 UI (추가 소요: 1~2일)

- **도형 속성 대화상자 확장**: 기존 PicturePropsDialog에 탭 추가
  - 테두리: 색상, 두께, 선 스타일
  - 채움: 없음/단색/그라디언트
  - 직선 전용: 화살표 시작/끝 모양, 크기
- **도형 내 텍스트 입력**: Rectangle/Ellipse에 TextBox 포함 시 더블클릭 → 텍스트 편집
  - 글상자 편집 모드 기존 코드 적용 가능

### Level 3: 고급 도형 편집 (추가 소요: 3~5일)

- **다각형 편집**: 꼭짓점 핸들 표시, 드래그로 이동, 추가/삭제
- **곡선 편집**: 베지어 제어점 핸들
- **호(Arc) 편집**: 시작각/종료각 핸들
- **코너 라운딩**: 사각형 모서리 둥글기 핸들
- **연결선($col)**: 모델/파서/렌더러 신규 구현 + 개체 간 연결 추적
- **그룹 개체**: 다중 선택(Shift+클릭), 그룹/해제 커맨드

### Level 4: 한컴 수준 완성 (별도 프로젝트 규모)

- 도형 카테고리 (기본, 말풍선, 별/배너, 순서도 등)
- 스마트 가이드 (정렬 보조선)
- 도형에 텍스트 흐름 연결
- 3D 효과, 그림자 편집
- 연결선 자동 라우팅

## 권장 사항

**Level 1을 현재 타스크 범위**로 진행하는 것을 권장합니다.

이유:
1. 기존 인프라(배치 모드, 핸들, 속성 API)가 완성되어 있어 새 코드 최소화
2. 직선/사각형/타원 3종으로 기본 도형 요구 충족
3. 리사이즈/이동/회전/삭제는 추가 구현 없이 동작
4. Level 2~3은 별도 타스크로 분리하여 점진적 확장 가능
