## 1단계: 전용 CSS 분리 + 대화상자 구조 개편

### 목표
- 인라인 스타일을 전용 CSS로 분리하여 유지보수성·시각 품질 향상
- 표 선택/셀 선택에 따라 탭 구성을 동적으로 변경

### 작업 내용

1. **`table-cell-props.css` 신규 생성**
   - 기존 `table-cell-props-dialog.ts` 내 인라인 스타일 전량 CSS 클래스로 이전
   - CSS 접두어: `tcp-` (table-cell-props)
   - 주요 클래스: `tcp-section`, `tcp-row`, `tcp-label`, `tcp-input`, `tcp-unit`, `tcp-btn-group`, `tcp-margin-grid`

2. **문맥별 탭 분기**
   - constructor에 `mode: 'table' | 'cell'` 파라미터 추가
   - `mode === 'table'` → 6탭 (기본/여백캡션/테두리/배경/표/셀)
   - `mode === 'cell'` → 4탭 (기본/여백캡션/표/셀) — 테두리·배경 탭 제외
   - 테두리/배경 탭에서 셀/표 전환 라디오 제거 (표 전용으로 단순화)

3. **대화상자 호출부 수정**
   - `input-handler-table.ts` 또는 호출 지점에서 현재 선택 상태(표/셀) 판별하여 mode 전달

### 검증
- 표 선택 시 6탭 표시, 셀 선택 시 4탭 표시
- 기존 기능 동작 유지

## 2단계: 표/셀 속성 탭 고도화 (표·셀·여백캡션)

### 목표
- 표 탭: "나눔" 3종 + 자동 경계선 설정 + "모두" 일괄 여백
- 셀 탭: "모두" 일괄 여백 + 한 줄로 입력 활성화 + 세로쓰기 눕힘/세움
- 여백/캡션 탭: 캡션 넣기 아이콘 그리드 + "모두" 일괄 조정 + 캡션 상세 옵션

### 작업 내용

1. **표 탭 보강**
   - 경계에서(Q): 라디오 2종 → 드롭다운 3종 (나눔/셀 단위로 나눔/나누지 않음)
   - 자동으로 나뉜 표의 경계선 설정(J): 체크박스 + 종류/굵기/색 하위 필드
   - "모두(A)" 스피너 버튼 추가 — 클릭 시 4방향 여백을 동일값으로 설정

2. **셀 탭 보강**
   - "모두(A)" 스피너 추가 (안 여백 일괄 조정)
   - "한 줄로 입력(S)" 체크박스 활성화 (disabled 해제)
   - 세로쓰기: "문 눕힘(Q)/문 세움(U)" 2종 아이콘 추가 (세로쓰기 체크 시 활성)
   - WASM API에 `singleLine` 속성 전달 필요 시 `table_ops.rs` 보강

3. **여백/캡션 탭 보강**
   - "모두(A)" 스피너 추가 (바깥 여백 일괄 조정)
   - 캡션 넣기: 드롭다운 → 3×3 시각적 아이콘 그리드 (8종 위치)
     - 위/아래/왼쪽(위·가운데·아래)/오른쪽(위·가운데·아래)
     - CSS 격자 + 아이콘 버튼으로 구현
   - 캡션 크기(S) mm 입력 추가
   - "여백 부분까지 너비 확대(W)" 체크박스 추가

### 검증
- 표 탭: 3종 페이지 분할 모드 설정/적용 확인
- 셀 탭: 한 줄로 입력 토글 동작
- 여백/캡션: 캡션 위치 8종 선택, "모두" 일괄 여백 동작

## 3단계: 표 테두리·배경 탭 고도화

### 목표
- 테두리 탭: 선 종류 시각적 격자 + 십자선 미리보기 + 자동 경계선 설정
- 배경 탭: 무늬 채우기 (면색+무늬색+무늬모양) 활성화

### 작업 내용

1. **테두리 탭 개편**
   - 선 종류(Y): `<select>` → 시각적 격자 (SVG 아이콘 또는 CSS 패턴)
     - 선 없음, 실선, 파선, 점선, 일점쇄선, 이점쇄선, 긴파선, 이중실선 등 12~16종
   - 미리보기 영역: CSS border → SVG 기반 십자선 프리뷰
     - 중앙 십자선(셀 구분선) 포함
     - 방향 버튼(L/A/R/B/O)을 프리뷰 상하좌우에 배치
   - "선 모양 바로 적용(I)" 체크박스 추가
   - 자동 나뉜 표 경계선 설정(J): 체크박스 + 종류/굵기/색 하위 필드
   - 안내 문구: "표 테두리는 [셀 간격]에 값을 입력해야 나타납니다" 추가

2. **배경 탭 고도화**
   - 색(Q) 라디오 선택 시 3개 필드 활성:
     - 면색(C): 컬러 피커
     - 무늬색(K): 컬러 피커
     - 무늬모양(L): 패턴 드롭다운 (없음/가로줄/세로줄/역슬래시/슬래시/십자/X자)
   - WASM `patternType`, `patternColor` 속성 읽기/쓰기 연결
   - 미리보기: CSS pattern 또는 SVG로 무늬 프리뷰 표시

### 검증
- 테두리 선 종류 시각적 격자에서 선택 후 미리보기 반영 확인
- 방향 버튼으로 개별 테두리 적용 확인
- 배경 무늬 채우기 설정 → 저장 → 렌더링 반영 확인

## 4단계: 셀 테두리/배경 별도 대화상자 + 컨텍스트 메뉴

### 목표
- 한컴의 "셀 테두리/배경" 별도 대화상자 (3탭) 신규 구현
- 컨텍스트 메뉴에 "셀 테두리/배경" 하위 메뉴 추가

### 작업 내용

1. **`cell-border-bg-dialog.ts` 신규 생성**
   - 3탭: 테두리 / 배경 / 대각선
   - **테두리 탭**:
     - 선 종류(Y) 시각적 격자 (3단계에서 구현한 컴포넌트 재사용)
     - 프리셋 버튼: 모두/바깥쪽/안쪽
     - 방향 버튼(L/A/R/B/O) + 미리보기
     - "선 모양 바로 적용(I)" 체크박스
     - 적용 범위: 선택된 셀(S) / 모든 셀(E) 라디오
   - **배경 탭**:
     - 3단계에서 구현한 배경 UI 재사용 (면색/무늬색/무늬모양)
     - 적용 범위: 선택된 셀(S) / 모든 셀(E)
   - **대각선 탭**:
     - 선 종류/굵기/색 (공통 컴포넌트)
     - `\` 대각선: 아이콘 버튼 8종 (G~8)
     - `/` 대각선: 아이콘 버튼 8종 (F~M)
     - `+` 중심선: 아이콘 버튼 4종 (N~Q)
     - 적용 범위: 선택된 셀(S) / 모든 셀(E)
     - 미리 보기
   - constructor에 `applyMode: 'each' | 'asOne'` 파라미터 (각 셀마다/하나의 셀처럼)

2. **컨텍스트 메뉴 연동**
   - 셀 선택 모드 우클릭 메뉴에 "셀 테두리/배경(B)" 하위 메뉴 추가
     - "각 셀마다 적용(E)..." → `CellBorderBgDialog(applyMode='each')`
     - "하나의 셀처럼 적용(Z)..." → `CellBorderBgDialog(applyMode='asOne')`

3. **WASM API 보강** (필요 시)
   - 대각선 속성 조회/설정: `getDiagonalProperties()`, `setDiagonalProperties()`
   - "각 셀마다 적용" vs "하나의 셀처럼 적용" 로직

### 검증
- 셀 블록 선택 → 우클릭 → "셀 테두리/배경" → 각 셀마다/하나의 셀처럼 선택 후 대화상자 열림
- 테두리/배경/대각선 각 탭 동작 확인
- 적용 범위 (선택셀/모든셀) 동작 확인
