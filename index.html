로컬 프라이버시를 지키는
오픈소스 HWP/HWPX 에디터

데이터 유출 걱정 없는 로컬 환경, HWP 호환성의 새로운 기준. BBDG HWP Editor로 당신의 폐쇄망 워크플로우를 최적화하십시오.

무료 다운로드 소스코드 보기

Native-Grade
Performance

Rust 코어와 WASM 런타임의 결합으로 브라우저 한계를 넘어서는 60fps 렌더링을 보장합니다. 대형 문서 로드 시에도 UI 프리징 없이 매끄러운 편집이 가능합니다.

Low-Latency Rendering

Air-Gapped Security

데이터가 로컬 샌드박스를 절대 벗어나지 않는 완전한 보안 환경을 제공합니다.

Zero-Leak Integrity

Seamless Flow

운영체제와 직접 소통하는 드래그 앤 드롭 시스템으로 직관적인 워크플로우를 완성했습니다.

Native OS Hooks

Versatile Delivery

MSI, Portable, NSIS 등 엔터프라이즈 환경이 요구하는 다양한 배포 규격을 한 패키지에 담았습니다. 실시간 버저닝을 통해 호환성을 보장합니다.

Multi-Target Builds
